CakeGame Engine

CakeGame Engine is a game engine written in Java powered by OpenGL. This multi-year project proved to be the single greatest learning experience I've had. It's goal was to be a full featured framework for building games. It's not "complete" in the sense that not every feature I originally wanted has been created. It provides a large number of features including asset management, window management, geometry packages, collision detection, vector and texture based text rendering, geometry rendering, a particle engine and more.

Context Free Grammar Parser and Language

This was a project done by me in PHP (of all things) utilizing a custom modified CYK algorithm at it's core, surrounded by a custom lexer and tree post-processor. It includes its own language for defining CFGs and spits out a parse tree that matches the structure of the original CFG definition. Unforunately it runs in O(n3) time so it's rather impractical for large amounts of text, but it proved to be yet another good learning experience. When I have the time I will add a demo page to this site where you can create CFGs and parse text and see the output parse tree.
